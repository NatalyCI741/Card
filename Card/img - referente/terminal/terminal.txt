->Para verificar que Node.js y npm están instalados:
node --version
npm --versión

->Si encuentras problemas con las dependencias:
npm install --legacy-peer-deps



Pasos para abrir y ejecutar el proyecto React
1. Navegar a la carpeta del proyecto:

cd nombre-de-tu-carpeta

Reemplaza "nombre-de-tu-carpeta" con el nombre real de tu directorio
2. Instalar las dependencias:

npm install

Este comando descarga e instala todas las dependencias listadas en el package.json

3. Iniciar el servidor de desarrollo:

npm start
